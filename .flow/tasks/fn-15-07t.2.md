# fn-15-07t.2 Ecosystem Hub Tab - Cross-project dashboard with GitHub links and status

## Description

Build the main Ecosystem Hub tab component at `components/ecosystem/ecosystem-hub-tab.tsx`. This is the central command center showing all three Agent Dashboards projects with live GitHub status, deployment links, and quick navigation.

### Implementation

**File:** `components/ecosystem/ecosystem-hub-tab.tsx`

**Layout:** 3 project cards in a grid, each showing:

- Project name + description
- GitHub link (opens in new tab)
- Deployment URL (Vercel for dashboards2.0, localhost for xano-v2-admin)
- Last commit: message, author, relative time (from `/api/ecosystem/github-status`)
- Branch info: current branches, default branch highlighted
- Tech stack badges (Next.js 16, React 19, etc.)
